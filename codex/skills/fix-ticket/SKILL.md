# Fix Skiff Ticket

Use when the user asks Codex to fix a Skiff ticket.

Read the ticket with Skiff MCP `get_ticket`, read assets with `get_ticket_assets`, and read the fix packet with `get_fix_packet` before editing. Confirm the linked repo matches the current local repo, propose a short plan, implement the fix, and summarize changed files plus verification.
